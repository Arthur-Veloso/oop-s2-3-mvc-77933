# VGC College Management System

## Features
- Role-based access (Admin, Faculty, Student)
- Course enrolment system
- Attendance tracking
- Assignment & exam results
- Result release control

## Technologies
- ASP.NET Core MVC
- Entity Framework Core
- SQL Server
- GitHub Actions (CI)
